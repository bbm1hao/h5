-webkit-appearance 会改变元素默认风格,利用这个特性 可制作自定义皮肤的 单多选按钮

Demo

快速制作

  • 准备好单选多选按钮图片
  • 清除checkbox 以及 radio 的默认样式
    input[type='radio'],input[type='checkbox']
    {
    -webkit-appearance : none;
    appearance : none;
    outline: none;
    border : 0 none;
    border-radius: 0;
    background-color:transparent;
    }
  • 自定义checkbox,radio样式
    .radio_checkbox_img
    {
    background-image: url("img/radio_checkbox.png");
    }
    .radio
    {
    width: 27px; height: 27px; background-position: -27px 0;
    }
    .radio:checked
    {
    background-position: -27px -27px;
    }
    .checkbox
    { width: 27px; height: 27px; background-position: 0 0;
    }
    .checkbox:checked
    {
    background-position: 0 -27px;
    }
  • html代码
    <input type="radio" class="radio_checkbox_img radio" name="one" checked/>
    <input type="radio" class="radio_checkbox_img radio" name="one" checked/>
    <input type="checkbox" class="radio_checkbox_img checkbox" name="ck" checked/>
    <input type="checkbox" class="radio_checkbox_img checkbox" name="ck" />
